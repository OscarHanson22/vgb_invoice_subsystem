# V(ery) G(ood) B(uilding) (Company) Invoice Subsystem
This project was for a class at UNL where we learned how to pair Java and SQL together, design projects/databases, and work with team members. 

The subsystem was made for invoicing products/equipment/contractors for the VGB Company. 

The class also required us to build a design document, which contains most of the information about this project.
